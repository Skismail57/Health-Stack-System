## Screenshot Guidelines:

- **Resolution**: 1920x1080 or 1280x720
- **Format**: PNG
- **File Size**: Compress to < 500KB each
- **Content**: Show actual application with realistic data
- **Quality**: High quality, clear interface elements

## How to Take Screenshots:

**Windows**: 
- Press `Windows + Shift + S` to open Snipping Tool
- Select area and save

**Mac**: 
- Press `Cmd + Shift + 4` and drag to select area
- Screenshot saves to Desktop

**Browser DevTools (for responsive)**:
- Press `F12` to open DevTools
- Click device toolbar icon
- Select mobile device
- Take screenshot

## Tips:

- Use browser in full screen mode
- Clear any personal/test data
- Ensure good lighting/contrast
- Show key features in action
- Include realistic sample data
- Crop unnecessary parts

## File Naming:

Use exact filenames as listed above. The README references these specific names.

Place all screenshots in this directory: `static/screenshots/`
